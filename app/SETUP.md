# GetFit App - Setup Instructions

## Install required dependency
Run this inside your my-getfit folder:

```bash
npx expo install expo-linear-gradient
```

## Files to copy into your project
Copy these files into your project's `app/` folder:
- index.tsx   → Welcome/Splash screen
- login.tsx   → Login screen
- register.tsx → Register screen
- home.tsx    → Calorie Dashboard
- log-meal.tsx → Log Meal screen

## Run the app
```bash
npm run start
```
Then scan the QR code with Expo Go on your iPhone.
